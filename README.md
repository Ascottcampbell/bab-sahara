# Bab Sahara

**The Gate to Morocco's Soul** — A beautiful, interactive web app for planning a personalized vacation to Morocco.

## Features

- **Dramatic Landing Page** — Stunning hero with high-quality Moroccan landscape photography and elegant typography
- **Interactive Trip Planner**
  - Choose trip duration (4–5 days up to 14+ days)
  - Select travel interests: Heritage cities, Sahara Desert, Atlantic Coast, Atlas Mountains, Culture & Cuisine, or Wellness
- **Smart Itinerary Generation** — Produces realistic day-by-day plans tailored to your choices
- **Transportation Guide** — Suggested methods to travel between destinations (trains, private drivers, 4x4s)
- **Travel Tips** — Culturally respectful, practical advice including desert packing, bargaining, and safety
- **Interactive Map** — Leaflet-powered map with your exact itinerary locations, route lines, and clickable popups
- **Print / PDF Export** — Beautiful printable version of your itinerary
- **Fully Responsive** — Works beautifully on desktop, tablet, and mobile

## How to Use

1. Open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari)
2. Scroll to the planner section or click **"Start Planning"**
3. Select your trip length
4. Choose the experiences that excite you most (multiple selections allowed)
5. Click **"Craft My Itinerary"**
6. Explore your personalized day-by-day plan, transportation recommendations, tips, and interactive map

**Pro tip:** On the planner screen, press the **B** key for an instant beautiful demo itinerary.

## Tech Stack

- Pure HTML + Tailwind CSS (via CDN)
- Vanilla JavaScript (no frameworks)
- Leaflet.js (interactive maps via CDN)
- Font Awesome icons
